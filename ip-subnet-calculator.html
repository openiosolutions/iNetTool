Understanding IP Subnetting

                
IP subnetting is the process of dividing a single, large network into smaller, more manageable sub-networks, or "subnets." This is a fundamental concept in computer networking used to improve performance, security, and organization.

                
                
What is CIDR Notation?

                
CIDR (Classless Inter-Domain Routing) notation is a compact way to represent a range of IP addresses. It combines an IP address with a slash (`/`) followed by a number. This number (from 0 to 32) represents the number of leading bits in the network portion of the address.

                
    
                    
  • Example: `192.168.1.1/24`
    • 
                    
  • The `/24` means that the first 24 bits are the network part, and the remaining 8 bits are for host addresses.
    • 
                


                
Key Terms in This Calculator

                
    
                    
  • Network Address: The very first address in a subnet, which identifies the network itself. It cannot be assigned to a device.
    • 
                    
  • Broadcast Address: The very last address in a subnet, used to send messages to all devices on that network simultaneously.
    • 
                    
  • Host Range: The range of usable IP addresses that can be assigned to individual devices (computers, phones, etc.) within the subnet.
    • 
                    
  • Subnet Mask: A 32-bit number that separates the network portion of an IP address from the host portion.
