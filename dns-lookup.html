What is DNS?

                
The Domain Name System (DNS) is the phonebook of the internet. Humans access information online through domain names, like `google.com`. Web browsers, however, interact through Internet Protocol (IP) addresses. DNS translates domain names to IP addresses so browsers can load internet resources.

                
                
Common DNS Record Types

                
A DNS lookup will show you various records associated with a domain. Here are some of the most common types:

                
    
                    
  • A Record: The "Address" record, which points a domain to an IPv4 address (e.g., `172.217.16.142`).
    • 
                    
  • AAAA Record: Similar to an A record, but it points to a more modern IPv6 address.
    • 
                    
  • CNAME Record: The "Canonical Name" record, which forwards one domain or subdomain to another domain. For example, `www.example.com` might point to `example.com`.
    • 
                    
  • MX Record: The "Mail Exchange" record, which directs email to a mail server.
    • 
                    
  • TXT Record: The "Text" record, which can hold arbitrary text. It's often used for verifying domain ownership, email security (SPF, DKIM), and other purposes.
    •
